# TTT4255-Gruppe21-H2025

## Prerequisites
- Node.js
- npm
- PlatformIO IDE

## How to run project
1. Navigate to /src/blunkesensor.ino and edit the SSID, pass, serveraddress and port variables (For the SSID and pass variable it would be recommended to use local env variables instead).
2. Upload the code to the arduino
3. Open a console/terminal in the /server folder and run `npm i`
4. After npm is done type `node server.js`
